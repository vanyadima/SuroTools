# Межсетевой экран Интернет Контроль Сервер

Учётнык данные для входа

| Интерфейс               | Способ подключения                                               | Логин                | Пароль по умолчанию          |
| :---------------------- | :--------------------------------------------------------------- | :------------------- | :---------------------------- |
| **Консоль управления**  | Непосредственно (монитор/клавиатура) или через SSH (порт 22)    | `recshell`           | `recovery`                    |
| **Меню первичной настройки** | При первом включении (после нажатия F2)                          | (не требуется)       | `recovery`                    |
| **Веб-интерфейс**       | Через браузер (порт 81, например `http://<IP-адрес>:81`)        | `root`               | `00000` (пять нулей)          |
